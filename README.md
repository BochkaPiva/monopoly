# PROLEUM Monopoly

Тестовый веб-стенд для мультиплеерной настольной игры «ПРОЛЕУМ: Сделка на миллион тонн».

## Запуск

```bash
npm run dev
```

Сайт будет доступен по адресу:

```text
http://127.0.0.1:5173/
```

## Что уже есть

- вход с ролями администратора и игрока;
- комнаты для тестовых партий;
- поле на 40 клеток;
- планшет рынка с ресурсами REG / PRM / DTL;
- логистика дня, открытые контракты и тендер;
- бросок 2D6 и журнал партии;
- раздел правил по ТЗ;
- админ-редактор игровых сущностей.
