# PROLEUM Monopoly: UI-направление

Визуальный язык должен ощущаться как премиальный нефтетрейдинговый стол, а не как обычная учебная веб-страница.

## Бренд-ориентиры

Из брендбука и референсов PROLEUM:

- глубокий черный как основной премиальный фон;
- чистый белый как рабочая поверхность;
- яркий желтый как главный акцент;
- точечная сетка 3x3 из логотипа;
- техно-схемы, линии, узлы, индустриальные пиктограммы;
- нефтетрейдинг, логистика, риск, хеджирование, поставка;
- плотная, но аккуратная модульная верстка.

## Принципы интерфейса

- Первый экран игры - сама партия, без лендинга.
- Левое меню скрыто в режиме игры.
- Карта крупная и остается главным объектом стола.
- В центре карты - рынок дня: цена, остаток, логистика, карта рынка.
- Команды игрока сгруппированы справа: ход, контракты, тендер, активные сделки.
- Ошибки правил объясняются прямо в disabled-кнопках и подсказках.
- Модалки используются для действий с последствиями: клетка, закупка, маршрут, событие.
- Планшеты игроков выезжают снизу как физические планшеты за краем стола.

## Следующий визуальный проход

1. Уменьшить белые панели, заменить их на черные игровые модули с желтыми маркерами.
2. Добавить “схемную” графику: тонкие линии, узлы, пунктирные маршруты вокруг центра поля.
3. Сделать карточки контрактов похожими на премиальные игровые карты: темный фон, тип, ресурс, доход, риск, срок.
4. Сделать маршруты визуальными: авто / ЖД / ЖД + нефтебаза как три варианта с мощностью и риском.
5. Добавить явный индикатор действия хода: бросок, клетка, коммерческое действие.
